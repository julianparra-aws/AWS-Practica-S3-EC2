# ☁️ AWS Cloud Project - EC2 + S3 Architecture

## 🚀 Overview
Este proyecto consiste en el diseño e implementación de una arquitectura básica en la nube utilizando servicios de AWS. El objetivo fue comprender el despliegue de infraestructura, la gestión de almacenamiento y la aplicación de buenas prácticas de seguridad en un entorno real.

## 🏗️ Architecture
La solución fue diseñada bajo un enfoque de infraestructura en la nube, incluyendo:

- Amazon EC2 como servicio de cómputo
- Amazon S3 para almacenamiento de objetos
- Configuración de red básica dentro del entorno AWS
- Security Groups para control de tráfico
- IAM para gestión de identidades y accesos

## ⚙️ Implementation
Durante el desarrollo del proyecto se realizaron las siguientes tareas:

- Despliegue de instancias en EC2 con acceso seguro mediante SSH
- Creación y configuración de buckets en S3
- Definición de reglas de seguridad para control de acceso (Security Groups)
- Configuración de permisos mediante IAM aplicando principio de mínimo privilegio
- Validación de conectividad y funcionamiento entre los servicios

## 🔐 Security Best Practices
Se implementaron buenas prácticas de seguridad como:

- Restricción de acceso por puertos específicos en Security Groups
- Uso de claves SSH para acceso seguro a instancias EC2
- Aplicación del principio de mínimo privilegio en IAM
- Separación de responsabilidades a nivel de accesos
